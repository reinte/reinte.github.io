Workshop: Deep Learning para Visión Computacional

En este workshop de Deep Learning para Visión Computacional, los estudiantes aprenderán los fundamentos de las redes neuronales convolucionales y su aplicación en problemas de visión computacional. Usaremos la biblioteca PyTorch para implementar modelos de Deep Learning y resolver problemas de clasificación de imágenes.
